# Проектная работа № 4. Mesto.

Перед вами четвертая проектная работа направления веб-разработки Яндекс-Практикума.
***
Данный проект представляет собой одностраничный сайт с фотографиями необъятной России. 
Вы можете внести свои данные, а в будущем, можно будет оценивать понравившиеся фотографии. 
***
Сайт аддаптивен, вы можете комфортно просмативать и взаимодействовать с формой, пользуясь телефоном. 
**В работе применяется:**
* flexbox-верстка;
* grig layout;
* позиционирование (относительное, абсолютное);
* продвинутая семантика;
* файловая структуру БЭМ — Nested;
* аддаптивная вёрстка (медиазапросы);
* базовый JS (взаимодействие с DOM-деревом, события)

***
**Системные требования:**
* HTML 5.0.;
* CSS3.0;
